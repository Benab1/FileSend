How to run:

- unpack the .tgz file
- change into the directory where the tar has been extracted to.

To run the program without gradle:
- run the runnable jar file with the command 'java -jar BBC_Exercise_BenBoreham-all.jar'
- to quit the application, enter 'quit'

To run the program and tests with gradle:
- change into the directory ./BBC_Exercise_BenBoreham
- using the gradle wrapper run a clean build with the command: 
  './gradlew clean build' (this should run all tests)

- the program can then be run with the command:
  './gradlew run' 


To view git history:

- The git repository is located in the main project directory (./BBC_Exercise_BenBoreham)
